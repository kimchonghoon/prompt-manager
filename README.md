# 나만의 프롬프트 관리기

GenAI 미션을 진행하며 여기저기 흩어진 프롬프트를, 한곳에 모아 관리하기 위한 파이썬 콘솔 프로그램입니다.
카테고리로 분류하고, 키워드로 검색하고, 자주 쓰는 것은 즐겨찾기로 관리할 수 있습니다.

## 실행 방법

Python 3.10 이상이 설치된 환경에서 아래 명령으로 실행합니다.

python app.py

메뉴가 나타나면 번호를 입력해 기능을 선택합니다. `0`을 입력하면 프로그램이 종료됩니다.

## 기능 목록

| 번호 | 기능 | 설명 |
|------|------|------|
| 1 | 프롬프트 추가 | 제목·내용·카테고리를 입력해 새 프롬프트를 등록 |
| 2 | 프롬프트 목록 | 저장된 모든 프롬프트를 번호와 함께 표시 |
| 3 | 카테고리별 조회 | 선택한 카테고리의 프롬프트만 모아 보기 |
| 4 | 프롬프트 검색 | 제목·내용에 키워드가 포함된 프롬프트 찾기 |
| 5 | 프롬프트 상세 보기 | 번호를 입력해 전체 내용 확인 |
| 6 | 즐겨찾기 관리 | 번호를 입력해 즐겨찾기 추가/해제 |
| 7 | 즐겨찾기 목록 | 즐겨찾기된 프롬프트만 모아 보기 |
| 0 | 종료 | 프로그램 종료 |

## 프롬프트 카테고리

프롬프트는 다음 6개 카테고리로 분류됩니다.

- **텍스트 생성** — 글쓰기, 카피, 문서 작성 관련 프롬프트
- **이미지 생성** — 이미지·썸네일 생성 관련 프롬프트
- **영상 생성** — 영상 스크립트·장면 구성 관련 프롬프트
- **페르소나** — 사용자·역할 설정 관련 프롬프트
- **자동화** — 노코드 자동화 워크플로 관련 프롬프트
- **기타** — 위 분류에 속하지 않는 프롬프트

## 데이터 구조

각 프롬프트는 리스트 안의 딕셔너리로 저장되며, 다음 정보를 가집니다.

- `title` — 제목
- `content` — 내용
- `category` — 카테고리
- `favorite` — 즐겨찾기 여부 (True/False)

## 참고

프로그램 실행 중에 추가한 프롬프트와 즐겨찾기 상태는 종료 시 초기화됩니다.
기본 프롬프트 3개는 이전 GenAI 미션에서 실제로 작성한 프롬프트로 등록되어 있습니다.
